# Guide simple - Trier les employés

## Comment trier les employés ?

### Méthode 1 : Menu déroulant (NOUVEAU ✨)

1. **Cliquez sur le bouton "Filtres & Tri"** en haut de la page

2. **Allez dans la section "Tri"** (en bas du panneau)

3. **Choisissez le champ** :
   - Matricule
   - Prénom
   - Nom
   - Email
   - Type de contrat
   - Date de début
   - Statut contrat
   - Secteur

4. **Choisissez l'ordre** :
   - Croissant (A → Z, 0 → 9, dates anciennes → récentes)
   - Décroissant (Z → A, 9 → 0, dates récentes → anciennes)

5. **C'est fait !** Les employés sont automatiquement triés

### Méthode 2 : Cliquer sur les colonnes

Cliquez sur l'entête de n'importe quelle colonne pour trier :
- **1er clic** : Tri croissant (↑)
- **2ème clic** : Tri décroissant (↓)

## Exemples pratiques

### Trier par nom de famille (A → Z)
```
1. Cliquez sur "Filtres & Tri"
2. Trier par : Nom
3. Ordre : Croissant
```

### Voir les employés les plus récents en premier
```
1. Cliquez sur "Filtres & Tri"
2. Trier par : Date de début
3. Ordre : Décroissant
```

### Trier par matricule (1, 2, 3...)
```
1. Cliquez sur "Filtres & Tri"
2. Trier par : Matricule
3. Ordre : Croissant
```

## Combiner filtres et tri

Vous pouvez combiner les filtres avec le tri :

**Exemple : CDI actifs triés par nom**
```
Section Filtres :
  • Statut : Actif
  • Type de contrat : CDI

Section Tri :
  • Trier par : Nom
  • Ordre : Croissant
```

## Voir le tri actuel

Quand le panneau est fermé, un petit indicateur montre le tri actif :

```
[↑] Tri : Prénom (A → Z)
```

## Réinitialiser

Pour tout remettre à zéro :
- Cliquez sur **"Réinitialiser tous les filtres"** en bas du panneau

---

**C'est tout !** Le tri fonctionne instantanément sans rechargement de page.
